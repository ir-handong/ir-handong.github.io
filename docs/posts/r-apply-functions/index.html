apply 함수란?

R에서 apply 함수 패밀리는 반복문(for loop) 없이 데이터에 함수를 적용할 수 있게 해주는 강력한 도구입니다.

1. apply() — 행렬/데이터프레임용

행 또는 열 단위로 함수를 적용합니다.

  • MARGIN = 1: 행 방향
  • MARGIN = 2: 열 방향
mat <- matrix(1:12, nrow = 3)
mat
     [,1] [,2] [,3] [,4]
[1,]    1    4    7   10
[2,]    2    5    8   11
[3,]    3    6    9   12
apply(mat, 1, sum)   # 행별 합계
[1] 22 26 30
apply(mat, 2, mean)  # 열별 평균
[1]  2  5  8 11

2. sapply() — 결과를 벡터로

nums <- list(a = 1:5, b = 6:10, c = 11:15)
sapply(nums, mean)
 a  b  c 
 3  8 13

3. lapply() — 결과를 리스트로

lapply(nums, mean)
$a
[1] 3

$b
[1] 8

$c
[1] 13

4. tapply() — 그룹별 집계

tapply(iris$Sepal.Length, iris$Species, mean)
    setosa versicolor  virginica 
     5.006      5.936      6.588

5. mapply() — 여러 인자를 동시에

mapply(rep, x = 1:4, times = 4:1)
[[1]]
[1] 1 1 1 1

[[2]]
[1] 2 2 2

[[3]]
[1] 3 3

[[4]]
[1] 4

한눈에 비교

함수 입력 출력 용도
apply 행렬 벡터/행렬 행/열 단위 연산
sapply 벡터/리스트 벡터 간단한 반복
lapply 벡터/리스트 리스트 안전한 반복
tapply 벡터 + 팩터 벡터 그룹별 집계
mapply 여러 벡터 벡터/리스트 다중 인자
