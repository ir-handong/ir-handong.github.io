로지스틱 회귀란?

로지스틱 회귀(Logistic Regression)는 종속변수가 범주형(0 또는 1)일 때 사용하는 분류 모델입니다.

  • 합격/불합격
  • 이탈/잔류
  • 질병 유/무

선형 회귀는 연속적인 값을 예측하지만, 로지스틱 회귀는 특정 범주에 속할 확률을 예측합니다.

핵심 개념: 시그모이드 함수

로지스틱 회귀는 선형 결합을 시그모이드 함수에 통과시켜 0~1 사이의 확률로 변환합니다.

\[ P(Y=1|X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 X)}} \]

# 시그모이드 함수 시각화
sigmoid <- function(z) 1 / (1 + exp(-z))

z <- seq(-6, 6, by = 0.1)
plot(z, sigmoid(z), type = "l", lwd = 2, col = "#0f3460",
     main = "시그모이드 함수", xlab = "z", ylab = "P(Y=1)",
     las = 1)
abline(h = 0.5, lty = 2, col = "gray50")
abline(v = 0, lty = 2, col = "gray50")
  • z가 클수록 → 확률이 1에 가까움
  • z가 작을수록 → 확률이 0에 가까움
  • z = 0일 때 → 확률 = 0.5 (결정 경계)

예제 데이터: 학생 합격 예측

시험 점수와 출석률로 합격 여부를 예측하는 모델을 만들어봅시다.

set.seed(42)
n <- 200

score <- round(rnorm(n, mean = 65, sd = 15), 1)
attendance <- round(runif(n, 50, 100), 1)

prob <- sigmoid(-8 + 0.08 * score + 0.05 * attendance)
pass <- rbinom(n, 1, prob)

students <- data.frame(
  점수 = score,
  출석률 = attendance,
  합격 = factor(pass, labels = c("불합격", "합격"))
)

모델 적합: glm()

R에서 로지스틱 회귀는 glm() 함수에 family = binomial을 지정하여 수행합니다.

model <- glm(합격 ~ 점수 + 출석률, data = students, family = binomial)

결과 해석

  • Estimate: 회귀 계수 (로그 오즈 단위)
  • Pr(>|z|): p-value. 작을수록 유의미
  • AIC: 모델 적합도 (낮을수록 좋음)

정확도, 정밀도, 재현율

accuracy <- sum(diag(confusion)) / sum(confusion)
precision <- confusion["합격", "합격"] / sum(confusion["합격", ])
recall <- confusion["합격", "합격"] / sum(confusion[, "합격"])
f1 <- 2 * precision * recall / (precision + recall)

ROC 곡선과 AUC

# 직접 ROC 계산
thresholds <- seq(0, 1, by = 0.01)
tpr <- sapply(thresholds, function(t) {
  pred <- ifelse(students$예측확률 >= t, 1, 0)
  actual <- as.numeric(students$합격 == "합격")
  sum(pred == 1 & actual == 1) / sum(actual == 1)
})
fpr <- sapply(thresholds, function(t) {
  pred <- ifelse(students$예측확률 >= t, 1, 0)
  actual <- as.numeric(students$합격 == "합격")
  sum(pred == 1 & actual == 0) / sum(actual == 0)
})

plot(fpr, tpr, type = "l", lwd = 2, col = "#0f3460",
     main = "ROC 곡선", xlab = "FPR (1 - 특이도)", ylab = "TPR (민감도)",
     las = 1)
abline(a = 0, b = 1, lty = 2, col = "gray50")

# AUC 근사 계산
auc <- sum(diff(sort(fpr)) * (tpr[order(fpr)][-1] + tpr[order(fpr)][-length(tpr)]) / 2)
legend("bottomright", legend = sprintf("AUC = %.3f", abs(auc)),
       col = "#0f3460", lwd = 2, bty = "n")
  • AUC = 1: 완벽한 분류
  • AUC = 0.5: 무작위 수준
  • AUC > 0.8: 일반적으로 좋은 모델

다중 클래스로 확장

로지스틱 회귀는 기본적으로 이진 분류이지만, 다항 로지스틱 회귀로 확장할 수 있습니다.

# iris 데이터로 3-class 분류
library(nnet)

iris_model <- multinom(Species ~ ., data = iris, trace = FALSE)

# 예측
pred <- predict(iris_model, iris)
table(예측 = pred, 실제 = iris$Species)
            실제
예측       setosa versicolor virginica
  setosa         50          0         0
  versicolor      0         49         1
  virginica       0          1        49
# 정확도
cat(sprintf("정확도: %.1f%%\n", mean(pred == iris$Species) * 100))
정확도: 98.7%

주의사항

1. 다중공선성 확인

# 독립변수 간 상관관계
cor(students[, c("점수", "출석률")])
              점수      출석률
점수    1.00000000 -0.03557505
출석률 -0.03557505  1.00000000

상관계수가 0.8 이상이면 다중공선성을 의심해야 합니다.

2. 선형성 가정

로지스틱 회귀는 독립변수와 로그 오즈 간의 선형 관계를 가정합니다.

3. 표본 크기

일반적으로 각 범주별 최소 10~20개 이상의 관측치가 필요합니다.
